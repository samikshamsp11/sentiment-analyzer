# AI-Powered Customer Feedback Sentiment Analyzer

This project is a beginner-friendly AI/ML application that analyzes customer feedback and predicts sentiment (Positive/Negative).

## Tech Stack
- Python
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression
- Streamlit

## Features
- Text preprocessing using TF-IDF
- Machine Learning pipeline
- Real-time sentiment prediction via UI
- Clean and simple architecture

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
